# OSI Model Explained

The OSI (Open Systems Interconnection) model defines how data travels through a network.

## 7 Layers of OSI Model

1. Physical  
2. Data Link  
3. Network  
4. Transport  
5. Session  
6. Presentation  
7. Application  

## Why It Is Important

- Helps understand networking
- Used for troubleshooting
- Standard communication model

## Example

When you open a website:

- Application layer sends request
- Transport layer handles data transfer
- Network layer routes packets
